Settings → Professional Reports → Default Style
    Pick the style applied by default for the company.

Settings → Technical → Professional Reports → Report Styles
    Create / edit / archive styles. Each style picks a template
    (Classic / Modern / Slim), theme colors, fonts, watermark and
    last-page PDF.

Contacts → (any partner) → Reports tab
    Override the company default style for this partner.

Invoice / Sales Order / Purchase Order / Delivery
    A *Professional Report Style* field lets you override the
    partner / company default for a single document.
